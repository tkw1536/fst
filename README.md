# fst - FTP File Sync Tool

A little script to keep in sync with an ftp server. Requirements: 

## Dependencies
* `lftp` - [sophisticated file transfer program](http://lftp.yar.ru/)
* Python 2 with modules: 
	* `sys`
	* `os`
            * `re`
	* `ConfigParser`
	* `json`
	* `shutil`
	* `subprocess`
